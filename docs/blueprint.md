# **App Name**: Apex Portfolio

## Core Features:

- Hero Section: Display name, title, tagline, and CTAs in a full-screen layout with a subtle animated background.
- About Me Section: Show a short bio, profile image, and key strengths.
- Skills Section: Categorize skills (Flutter, Dart, APIs, etc.) with animated progress bars.
- Projects Section: Display projects in a grid with titles, descriptions, and links to GitHub and live demos. Use generative AI to generate the project description based on title and tech stack; the AI will act as a tool that may be overridden.
- Experience/Education Timeline: Show experience and education in a vertical timeline layout.
- Contact Section: Include a minimal contact form and social links.
- Footer: Display copyright information and quick navigation links.

## Style Guidelines:

- Background color: Charcoal (#121212) for a dark-mode-first design.
- Primary color: Electric blue (#7DF9FF) to give a premium look, and contrast well with the dark background.
- Accent color: Purple (#BE52F2) for subtle highlights, with slightly less saturation and brightness than the primary to achieve sufficient contrast.
- Body and headline font: 'Inter', a sans-serif with a modern look.
- Use minimalistic icons from a set like Material Icons or Font Awesome.
- Employ a minimalist layout with generous spacing, rounded cards, and soft shadows for a premium look.
- Use smooth page transitions, section fade-ins/slide-ups on scroll, and button hover animations.
